# Credit Scoring Project

## Description
Ce projet a pour objectif de développer un **modèle de credit scoring** permettant de prédire la probabilité de défaut de paiement des clients. Il inclut :

- Préparation et nettoyage des données (`src/data_prep.py`)
- Création et entraînement des modèles (`src/model_utils.py`)
- Évaluation des performances (`src/metrics.py`)
- Analyse d’explicabilité (`src/explainability.py`)
- Exploration et visualisation des données (`notebooks/data.ipynb`)
